# jogos-pico8
Um local pra juntar os códigos de pico 8 que eu escrevo. Foram feitos pela education edition

Como rodar o código:
1. baixe o arquivo que você queira jogar
2. Abra o Pico 8 Education Edition
3. Digite "load" no terminal. Isso abrirá o explorador de arquivo
4. Localize o arquivo e selecione
5. Digite "run" ou CTRL + R para rodar o código.
